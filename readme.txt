== Description ==

A widget for displaying the most recent twitter status updates for a particular hash tag.

== Installation ==

1. Upload the `twitter-hash-tag-widget` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer.
1. Activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer.

== Upgrade ==

1. Use the plugin updater in WordPress or...
1. Delete the previous `twitter-hash-tag-widget` folder from the `/wp-content/plugins/` directory
1. Upload the new `twitter-hash-tag-widget` folder to the `/wp-content/plugins/` directory

== Usage ==

1. Install.
1. Activate.
1. Add to your active widgets at `Appearance->Widgets` and configure.
